Mini tutoriel Angular

  • Vérifier les versions: ng --version || Npm --version || Node --version
  • Créer un projet complet avec router: ng n [nom] --routing
  • Installation du serveur json (-g pour global || @version pour un version précise du json-server): npm i -g json-server || npm i -g json-server@[version]
  • Installation de bootstrap: npm i -g bootstrap || npm I -g bootstrap @[version]
  • Installation de fortawesome: npm I @fortawesome/ fontawesome-free –save
  • Installation jquery: npm I jquery --save
  • Désinstaller un fontawsome: npm uninstall @fortawesome/angular-fontawesome@latest
  • Réparer les conflits de librairies: npm audit fix
  • Choix et installation des composants: ng g c composants/[nom]
  • Création d’un dossier à la source plus création fichier: Mkdir database && cd database && touch data.json
  • Création service: ng g s services/[nom]
  • Lancement du site: npm start
  • Lancement json-server: json-server -w .\data.json --port [4000]
  • Lancement de test unitaire(par logiciel de navigation): ng test
  • Lancement des specs: ng lint
  • Lancement de test et lint: ng e2e
